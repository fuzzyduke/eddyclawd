# eddyclawd

Our custom implementation of Clawdbot (OpenClaw), designed for private VPS hosting.

## Project Goals
- Personal AI assistant for automation and privacy.
- Hosted on a private VPS (independent of the Vara Contabo VPS).
- Experimental features and custom skill integrations.

## Tech Stack (Planned)
- Node.js / TypeScript
- Git for version control
- Docker (for VPS deployment)

## Hosting
- **Target**: Private VPS
- **Status**: Initialization Phase
